Display a gallery of animated posts unsing a shortcode for theme Gridsby.

== Description ==

Gallery of Animated Posts is a simple plugin for theme [Gridsby](https://modernthemes.net/wordpress-themes/gridsby/) that adds a shortcode for displaying a gallery of animated posts anywhere in the content of a page or a post. Posts to be displayed are of post format *image* and can be selected by tag and query parameters as described in [Class Reference/WP Query](https://codex.wordpress.org/Class_Reference/WP_Query).

= Basic Usage Example =

`[gallery-of-animated-posts]`

will output the HTML code for a gallery of all *image* posts and

`[gallery-of-animated-posts tag="frowny"]`

for a gallery consisting only of *image* posts that are tagged with the slug *frowny*.

== Installation ==

Gallery of Animated Posts is installed just as any other plugin. For more information see [Installing Plugins](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

== Frequently Asked Questions ==

= Is Gallery of Animated Posts free? =

Yes! The plugin is Open Souce and always will be free. 

= Can I use the plugin with a theme different from Gridsby? =

No! Gallery of Animated Posts is specifically designed to enhance theme Gridsby by adding more flexibility when displaying galleries of *image* posts.
